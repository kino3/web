●［またさっさと引越し！ 2］全財産とは何か

引越しというのは、自らの財産のありようを、自らに突きつける行為である。強烈な自覚を促す。
　
結局のところ、「もの」は運ぶ必要があるのである。

で、自分の財産（お金は銀行にあるので、固定資産とでも言うのか）は、以下のようなものである。
== 衣類 ==
＊衣服
＊靴
＊かばん

== 書籍、文具 ==
＊書籍
＊写真や手紙などの思い出
＊必要書類（保険とか金融とか）
＊その他雑多な小物

== 大型 ==
＊家具
＊家電
＊パソコン

こういったものをひとつずつ検討し、不要なものは捨てる。
おそらく、衣類と書籍は大量に処分できると思う。
目標は、衣類は衣装ケース１つ、書籍は100冊。いや、50冊でいいか？
いまから２，３か月かける。

●［またさっさと引越し！ 1］また捨てる

別に引越しが好きということはないのだが、また引越しをする羽目になりそうだ。家賃を軽減することが主な目標で、今回は都内の近所に越したい。
　
そして、これはぼく個人の目標なのだが、ぼちぼち奈良に戻りたいので、これを機に荷物を整理しておきたいのである。前回捨て切れなかったものを捨てる。

とりあえず、いまだ開封すらおぼつかないダンボールの中身は、思い出と共に葬り去ることになるのだろうと思う。以下次号。
