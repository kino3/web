●［またさっさと引越し！ 2］全財産とは何か

引越しというのは、自らの財産のありようを、自らに突きつける行為である。強烈な自覚を促す。
　
結局のところ、「もの」は運ぶ必要があるのである。

で、自分の財産（お金は銀行にあるので、固定資産とでも言うのか）は、以下のようなものである。
== 衣類 ==
＊衣服
＊靴
＊かばん

== 書籍、文具 ==
＊書籍
＊写真や手紙などの思い出
＊必要書類（保険とか金融とか）
＊その他雑多な小物

== 大型 ==
＊家具
＊家電
＊パソコン

こういったものをひとつずつ検討し、不要なものは捨てる。
おそらく、衣類と書籍は大量に処分できると思う。
目標は、衣類は衣装ケース１つ、書籍は100冊。いや、50冊でいいか？
いまから２，３か月かける。

まあ、きっと、生駒山上から見た方がずっときれいなんだろうが、そういうムーディな事件には遭遇したことがないので知らない。

●転向生（古本セール）

引越しに関係する事務手続きというのは結構たくさんあって、チェックリストのようなものがネットにも出回っている。僕も多分に漏れずそのうちの一つを使わせていただいているのだが、項目のなかに「転校手続き」というのがあった。

今でこそ微妙に仕事場を転々としているが、僕は一度も転校したことがない。実家にいたときは幼稚園から小学校まで8年と、中高6年それぞれ同じところに通い続けた。東京に来てからも5年間寮暮らしだったし、よく考えると、ある組織を区切りと関係なく離脱した経験があまりない。

よく、転校生は教室入って最初が勝負、みたいな話を聞くが、どんな気分なんだろうか。

僕は奈良でも結構なニュータウンに住んでいたので、転入生はとてもたくさん見てきたのだが、いまいちそのときの雰囲気を思い出せない。妙な記憶力だけはあって、いまでも小学校のアルバムを見たら、誰が何年生ぐらいに転校してきたとか、こいつは幼稚園から一緒やとかほぼ全て言えるにも関わらず、である。見ず知らずの少年少女が、集団の仲間として受容される瞬間には、何があるのだろう。「今日からお前は友達やで」的な何かがあるのか。

形式ばかり気にして、本質をつかめないのは昔からなんだなと思う。引越しを機に転校、いや転向したい。

●［さっさと引越し！ 1］街と家

次に引越すときは、奈良に帰るときやと思ってたのに、また東京に戻るというのは、不思議な気分です。

そこそこ気に入ってたこの街ともあと１か月でお別れです。いいとこでテレビを消されたような気分。

新しい街。それはそれで楽しみなんやけど。なくして気づく何とやら、っちゅう感じやね。

ま、今さらぼくが言うことでもないんですが、諸行無常。せっかくやし、と意味不明な理由をつけて、いろんなものを捨ててみます。目指すは、「捨てようという思いすら捨てよ」と言った一遍上人か。

記憶力がいいと、楽しいことも多いけど、悲しいことも減らんね。
