ナビゲーション・メニュー

ナビゲーション・メニューはすべての画面の上部にあり、ここからMovable Typeシステムの主要な画面と自身のアカウント情報にアクセスできます。

  • メイン・メニュー：利用可能なブログの一覧を表示します。
  • システム・メニュー：システム全体の管理画面を表示します。
  • ヘルプ：Movable Typeユーザー・マニュアルを表示します。
  • 「ブログ」プルダウン・メニュー：直接、指定したブログの管理画面を表示します。
  • ユーザー名：現在ログインしているユーザーのプロフィール・ページを表示します。
  • ログアウト：ログアウトし、初期画面(ログイン画面)を表示します。
  • サイトの確認：現在操作しているブログの公開先を表示します。ブログを公開していない場合、このリンクは表示されません。

パンくずリスト

パンくずリストは、メイン・ページから、現在表示しているページまでの履歴を、途中経過したページも含め、階層的に表示します。パンくずリストに表示されるページはどれもリンクされているので、そのどれか1つに簡単に戻れます。各画面のタイトルのすぐ上にあります。

タブ

タブは、同じ画面に属する異なるページを本のページのように表示するのに使うナビゲーション・コントロールです。タブは、1一つの画面に属する全ページを平面状に並べると情報が混雑して見える場合に使います。

フィルター

フィルターは、ユーザーが一覧表示のアイテムについて、条件を絞り込んで表示するナビゲーション・コントロールです。たとえば、エントリーの一覧表示で、「下書き」状態のエントリーのみを表示できます。フィルターを指定可能な画面では、画面のタブの直下の「フィルター」リンクをクリックすると、フィルター・コントロールを表示します。右側には「クイック検索フィルター」リンクがあり、よく使う条件をあらかじめリスト登録しています。リストから選ぶだけの簡単な操作で絞り込みができます。

表示設定

「表示設定」は、一覧表示の件数や並べ替えの順序等をカスタマイズできるダイナミックなポップアップ画面です。「表示設定」は、一覧を表示する画面の下部にある「画面の表示設定を変更」リンクをクリックすると、表示されます。カスタマイズできる内容は、一覧画面によって異なります。

ブログ・ツールバー

ブログ・ツールバーは、ボタンが縦一列に並んだバーで、ここから、現在選択したブログについて、ユーザーが利用できる機能にアクセスできます。このツール・バーは、個々のブログで利用できる機能の一覧で、全画面の左側にあります。

  • 新規エントリー：エントリーを新しく投稿します。
  • エントリー：ユーザーが編集可能なエントリーの一覧を表示します。
  • ファイルをアップロード：ポップアップウィンドウが開き、画像、音声等のファイルをブログにアップロードできます。また、アップロードしたファイルのリンクを含めたエントリーを投稿したり、サムネイルも作成できます。
  • コメント：特定のブログ宛てのすべてのコメントを管理できます。
  • コメント投稿者：コメント投稿者を管理するための一覧です。
  • 通知：エントリーを公開したときのお知らせメールの宛先の一覧です。
  • トラックバック：特定のブログ宛てのすべてのトラックバックを管理できます。
  • テンプレート：特定のブログを構成するテンプレート、テンプレート・モジュールを管理できます。
  • カテゴリー：既に登録しているカテゴリーを管理できます。
  • 設定：特定のブログについて、URLや出力ファイル名等の各種の環境設定ができます。
  • 検索：特定のブログを対象に、文字列の検索ならびに置換ができます。
  • ログ：特定のブログに対するアクションのログを一覧表示します。ログをCSVファイル形式で保存することもできます。
  • 読み込み/書き出し：他のツールで作成したエントリー情報を登録したり、既存のエントリーをテキスト・ファイルに書き出すことができます。
  • サイトの再構築：ブログ・サイトの再更新のためのポップアップ画面を表示します。
  • サイトの確認：別のブラウザー・ウィンドウが開き、指定したブログのホームページを表示します。

システム・ツールバー

システム・ツールバーは、ボタンが縦一列に並んだバーで、システム全体を管理するためのツールにアクセスできます。このツールバーは、各システム・メニュー画面の左側にあります。

  • ブログ：現在、登録しているすべてのブログの一覧を表示します。また、新規にブログを作成することもできます。
  • 投稿者：現在、登録しているすべての投稿者の一覧を表示します。また、新規に投稿者を登録することもできます。
  • プラグイン：現在、登録しているプラグインの一覧を表示します。また、各プラグインの環境設定もできます。
  • エントリー：すべてのブログのエントリーを一覧表示します。
  • コメント：すべてのブログへのコメントを一覧表示します。
  • トラックバック：すべてのブログへのトラックバックを一覧表示します。
  • 設定：システム全体の設定ができます。
  • 検索：すべてのブログを対象に、文字列の検索ならびに置換ができます。
  • ログ：システム全体のアクションのログを一覧表示します。ログをCSVファイル形式で保存することもできます。

メイン・メニュー

メイン・メニューはログインした直後に表示される画面です。この画面からリンクやナビゲーション・メニューを使って、Movable Typeシステムの他の機能を利用できます。また、この画面には、各画面の上部にあるナビゲーション・メニューの「メイン・メニュー」リンクをクリックすると、どの画面からもアクセスできます。

  • ナビゲーション・メニュー：システムとアカウントのへアクセスできます。
  • ブログの一覧：ユーザーがアクセスできるブログを一覧表示し、個々のブログの機能へのショートカットを表示します。
  • Movable Typeニュース：シックス・アパートからのMovable Typeに関連したニュースのヘッドラインを一覧表示します。
  • システム全体の機能：システム全体の各機能にアクセスできます。
  • クイック投稿：１クリックでブログへの投稿ができるブックマークレットを作成します。

ブログ・メニュー

この画面から、指定のブログ内のエントリーやコメント、トラックバックにアクセスできます。また、ブログの環境設定ができます。この画面は、画面上部のナビゲーション・メニューにある「ブログ」プルダウン・メニューからアクセスできます。

  • ブログ・ツールバー：ボタン操作で、現在のブログの利用可能なコマンドを実行できます。
  • パンくずリスト：メイン・ページから、現在表示しているページまでの履歴を、途中経過したページも含め、階層的に表示します。
  • スタート・ページ：現在のブログに関する説明文を表示します。この説明文は、個々のブログでカスタマイズできます。
  • 最近のエントリー：最新のエントリー5件を表示します。
  • 最近のコメント：最新のコメント5件を表示します。
  • 最近のトラックバック：最新のトラックバック5件を表示します。

システム・メニュー

「システム・メニュー」画面から、システム全体のエントリー、コメント、トラックバック、リソース、および環境設定画面にアクセスできます。この画面は、画面上部のナビゲーション・メニューにある「システム・メニュー」リンクからアクセスできます。

  • システム・ツールバー
  • Movable Typeニュース：シックス・アパートからのMovable Typeに関連したニュースのヘッドラインを一覧表示します。
  • システムの状況：ブログ数、投稿者数、エントリー数等の現在のシステムの数値を表示します。
  • リンク集：ユーザー登録、ユーザー・サポート等の役立つリンク集です。
